# SQL Queries - Movies Database

This folder contains a detailed report of 50 SQL queries performed on a comprehensive Movies database schema including actors, directors, genres, and ratings.

## Modules Covered

### 1. Basic Queries (1-10)
Focuses on simple data retrieval, filtering by year/title, and basic pattern matching (LIKE).

### 2. Sub-queries (11-26)
Demonstrates the use of nested SELECT statements to find specific actors, directors, or movies based on internal logic.

### 3. Joins (27-50)
Advanced data retrieval using INNER JOIN and LEFT JOIN to connect:
- Movie Cast and Actors
- Movie Direction and Directors
- Movie Genres and Genres
- Movie Ratings and Reviewers

## Key Technical Skills
- Multi-table Joins
- Aggregate Functions (MAX, MIN, AVG, COUNT)
- Data Grouping (GROUP BY, HAVING)
- Set Operations (UNION)
- Sub-query nesting
