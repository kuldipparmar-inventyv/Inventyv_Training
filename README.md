# Inventyv_Training

This repository contains the tasks completed during my Inventyv training program. 

---

## Repository Structure

### 01_Logic_Building
* **Description:** Focuses on computational logic and problem-solving.
* **Contents:** Flowcharts for various problems to visualize program logic and structure.

### 02_HTML_CSS_Task
* **Description:** Development of a responsive authentication interface using HTML and CSS.
* **Features:**
    * Includes Login and Signup forms.
    * Implements a swap mechanism between forms using pure CSS.
    * Fully responsive design for mobile and small-screen devices.
* **Technologies:** HTML, CSS.

### 03_Sql_Queries
* **Description:** Leaarning of Sql through 50 sql Quries.
* **Contents:** 50 SQL queries based on a Movies database.
* **Details:** Each query includes the written SQL code along with the resulting data output.

### 04_JavaScript_Tasks
* **Description:** Core tasks focused on asynchronous logic, logical thinking, and Symbols usecase.
* **Contents:**
    * **01_array_promise_problem.js**: Implementation of array merging using spread syntax and data validation via Promises.
    * **02_numerical_pattern.js**: A logic-heavy exercise generating coordinate pattern using the bitwise identity `-(~(n))` instead of standard addition.
    * **03_first_task_with_symbols_problem.js**: Refactor of array tasks using `Symbol` identifiers to ensure unique and encapsulated object keys.

### 05_Exercism_JS
* **Description:** Solutions for the [Exercism JavaScript Track](https://exercism.org/tracks/javascript) focused on ECMAScript standards.
* **Contents:** Over 25 exercises covering basic syntax, Boolean logic, closures, memoization, and Regex pattern matching etc. to understand different javascript concepts.
