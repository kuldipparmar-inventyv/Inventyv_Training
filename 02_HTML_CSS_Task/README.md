# HTML and CSS Authentication Task

This folder contains a webpage focused on building a user authentication interface without the use of JavaScript.

## Features
- **Login and Signup forms:** Clean and functional user interfaces.
- **Pure CSS Toggle:** Implemented a mechanism to swap between the login and signup forms using only CSS techniques.
- **Responsive Design:** The layout is optimized for small screen devices and mobile phones.
